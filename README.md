# blog-vue
## 简介
个人博客的前端项目，主要用到的技术和插件如下：
- vue
- element-ui
- vue-router
- vue-axios
- mavon-editor
- highlight.js
- marked.js
- ......

目前已具备功能(持续开发中...)：
- 登录页面
- 个人主页
- 文章管理
- 文章编辑
- 文章预览
- ......

本人前端技术一般，如果哪位大佬路过看见，欢迎留下建议和批评。

npm install
